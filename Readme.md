# Systematic Mapping Study on Schema Evolution

## Replication package

The repository contains scripts and all data referenced in the paper "A Systematic Mapping Study in the area of Schema Evolution". It serves as a replication package (a) for collection of all publications and bibliographic metadata contained in the analyzed corpus, and (b) for reproducing all figures and results contained in the paper. 

## Directory structure

The directory structure of the repository is as follows:

- `corpus`: bibliographic metadata of the corpus in various formats,
- `scripts`: includes scripts for downloading and processing bibliographic metadata from DBLP and SCOPUS.
- `dblp-results`: raw search results in JSON and BIB format as downloaded from DBLP.
- `scopus-results`: raw search results in CSV format as downloaded from SCOPUS 
- `process`: all data artifacts produced during compilation, screening, analysis and review of the corpus
